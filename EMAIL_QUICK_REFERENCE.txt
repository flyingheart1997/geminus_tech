## 30-Second Setup

```bash

```

## API Endpoint

**URL:** `POST /api/contact`

**Request:**
```json
{
  "name": "John Doe",
  "email": "john@example.com",
  "message": "I would like to know more about your services"
}
```

**Success Response (200):**
```json
{
  "success": true,
  "message": "Your inquiry has been sent successfully..."
}
```

**Error Response (400):**
```json
{
  "error": "Invalid email format"
}
```

## Troubleshooting Checklist

- [ ] .env.local file exists
- [ ] EMAIL_USER and EMAIL_PASSWORD are set
- [ ] Using Gmail App Password (not regular password)
- [ ] 2-Step Verification enabled on Gmail
- [ ] Dev server restarted after .env changes
- [ ] Check browser console for errors
- [ ] Check terminal for server errors
- [ ] Check spam folder for emails

## Common Issues

| Issue | Solution |
|-------|----------|
| "Not configured" | Restart dev server after adding .env.local |
| "Failed to send" | Verify Gmail credentials, check 2FA |
| No emails | Check spam folder, verify recipient email |
| Slow submission | Email service latency (normal 2-5 sec) |
